# <span style="font-size:15pt;color:BurlyWood">문제 풀이</span>

DP 문제로 기존에 2차원 배열을 하던 것을 3차원 배열로 확장시킨 문제로, 비트연산을 잘 활용하면 문제 해결이 가능합니다. 또한 오버플로우를 신경 써줘야합니다.
<br>

|`알고리즘`|`시간복잡도`|`공간복잡도`|
|:---:|:---:|:---:|
| DP | $O(N * 1024 * 10)$| $O(N * 1024 * 10)$ |
